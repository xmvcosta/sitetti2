Firewall - Gerenciamento de Internet

FIREWALL é o nome dado ao dispositivo de uma rede de computadores que tem por função regular o tráfego de rede entre redes distintas (internet) e impedir a transmissão e/ou recepção de dados nocivos ou não autorizados de uma rede a outra. É utilizado para evitar que o tráfego não autorizado possa fluir de um domínio de rede para o outro. O termo inglês firewall faz alusão comparativa à função que desempenha para evitar o alastramento de dados nocivos dentro de uma rede de computadores, da mesma forma que uma parede corta-fogo (firewall) evita o alastramento de incêndios pelos cômodos de uma edificação. Existe na forma de software e hardware ou na combinação de ambos. A instalação depende do tamanho da rede, da complexidade das regras que autorizam o fluxo de entrada e saída de informações e do grau de segurança desejado.
